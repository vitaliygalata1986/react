1. Frontend Setup

   ```sh
   npm install
   ```

   1.3 Start the development server:

   ```sh
   npm run start
   ```

2. Backend Setup

   2.1 Open a terminal and navigate to the api directory:

   ```sh
   cd api
   ```

   2.2 Install dependencies:

   ```sh
   npm install
   ```

   2.3 Start the backend server:

   ```sh
   npm run dev
   ```
